# Veridaq Brand Guidelines

## Brand Overview

Veridaq is a professional compliance technology platform specializing in KYC (Know Your Customer) and AML (Anti-Money Laundering) solutions for EU businesses. The brand positioning emphasizes reliability, European compliance expertise, and practical business solutions.

## Visual Identity

### Logo
- **Primary Logo**: Veridaq wordmark with transparent background
- **File**: `/public/images/veridaq-logo-transparent.png`
- **Usage**: Main navigation, footer, authentication pages
- **Sizing**: Standard height 32px for headers, 40px for standalone usage

### Color Palette

#### Primary Colors
- **Brand Orange**: `#ff8173` (rgb(255, 129, 115))
  - Primary CTA buttons, links, highlights
  - Warm, approachable, professional
  - Use for: Primary actions, brand accents, interactive elements

#### Secondary Colors
- **Navy Accent**: `#011949` (rgb(1, 25, 73))
  - Headers, important text, secondary buttons
  - Professional, trustworthy, authoritative
  - Use for: Headings, navigation, professional contexts

#### Neutral Palette
- **White**: `#ffffff` - Primary background
- **Light Gray**: `#f8fafc` - Section backgrounds
- **Medium Gray**: `#64748b` - Body text
- **Dark Gray**: `#1e293b` - Headings, important text

#### Functional Colors
- **Success**: `#16a34a` - Confirmations, positive states
- **Warning**: `#ca8a04` - Alerts, cautions
- **Error**: `#dc2626` - Errors, critical alerts

### Typography

#### Font Stack
- **Primary**: Inter (system-ui fallback)
- **Display**: Space Grotesk for headings (Inter fallback)
- **Monospace**: JetBrains Mono for code

#### Font Weights
- **Regular** (400): Body text, descriptions
- **Medium** (500): Emphasized text, labels
- **Semibold** (600): Headings, important text
- **Bold** (700): Major headings only (use sparingly)

#### Hierarchy
- **H1**: 2.5rem (40px), font-semibold
- **H2**: 2rem (32px), font-semibold  
- **H3**: 1.5rem (24px), font-medium
- **Body**: 1rem (16px), font-normal
- **Small**: 0.875rem (14px), font-normal

## Design Principles

### Visual Style
- **Clean & Minimal**: Avoid visual clutter, focus on content
- **Professional**: Business-appropriate, trustworthy appearance
- **Accessible**: High contrast ratios, readable typography
- **Consistent**: Uniform spacing, consistent component styling

### Iconography
- **Minimal Usage**: Avoid generic icons (shields, arrows, checkmarks)
- **Simple Shapes**: Use dots, lines, basic geometric forms when needed
- **Purposeful**: Only include icons that add clear functional value
- **Brand Consistent**: Match overall clean, professional aesthetic

### Layout & Spacing
- **8px Grid System**: All spacing based on 8px increments
- **Generous Whitespace**: Allow content to breathe
- **Consistent Margins**: 
  - Sections: 80px (py-20) vertical spacing
  - Components: 24px (mb-6) standard spacing
  - Elements: 16px (mb-4) tight spacing

## Content Guidelines

### Tone of Voice
- **Professional**: Business-appropriate language
- **Factual**: Avoid superlatives and unsubstantiated claims
- **Clear**: Direct communication without jargon
- **Helpful**: Focus on practical value and solutions

### Messaging Principles
- **No Performance Claims**: Avoid "10x faster," "90% reduction" etc.
- **No Superlatives**: Remove "revolutionary," "amazing," "incredible"
- **Factual Benefits**: Focus on actual features and capabilities
- **EU Focus**: Emphasize European compliance and regulatory alignment

## Component Standards

### Buttons
- **Primary**: Brand orange background, white text, rounded corners
- **Secondary**: White background, navy border, navy text
- **Sizes**: Standard padding (px-6 py-3), consistent across site

### Cards
- **Background**: White with subtle border
- **Shadow**: Minimal drop shadow for depth
- **Corners**: Rounded (rounded-xl)
- **Padding**: Consistent internal spacing (p-6 or p-8)

### Navigation
- **Clean**: Minimal styling, clear hierarchy
- **Consistent**: Same styling across all navigation elements
- **Accessible**: Clear focus states, proper contrast

## Implementation Notes

### CSS Classes
- Use Tailwind utility classes consistently
- Avoid custom CSS unless absolutely necessary
- Maintain consistent spacing using the 8px grid system

### Responsive Design
- Mobile-first approach
- Consistent breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px)
- Maintain design integrity across all screen sizes

### Performance
- Optimize images for web delivery
- Use appropriate image formats and sizes
- Minimize CSS and JavaScript bundle sizes

## Brand Applications

### Website Sections
1. **Hero**: Clean introduction with clear value proposition
2. **Features**: Straightforward feature presentation
3. **Industries**: Practical industry applications
4. **Benefits**: Factual advantages without claims
5. **Contact**: Professional contact form and information

### Content Types
- **Blog Posts**: Professional, educational content
- **Product Pages**: Clear, factual feature descriptions
- **Case Studies**: Real examples without exaggerated results
- **Documentation**: Clear, helpful technical content

## Quality Checklist

Before publishing any design updates:

- [ ] Colors match brand palette (orange primary, navy accent)
- [ ] Typography uses appropriate weights (avoid font-black)
- [ ] No generic or overused icons
- [ ] No performance claims or superlatives
- [ ] Content is concise and valuable
- [ ] Consistent spacing using 8px grid
- [ ] Accessible contrast ratios maintained
- [ ] Mobile responsive design verified
