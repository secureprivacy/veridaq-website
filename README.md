# Veridaq Compliance Platform

This project is a React-based compliance platform built with Vite, TypeScript, and Supabase.

## Technology Stack
- **Frontend**: React, TypeScript, Vite
- **Styling**: TailwindCSS
- **Backend/Database**: Supabase
- **Environment**: Bolt.new (Base), Antigravity (Supporting)

## Getting Started

### Prerequisites
- Node.js (>=18.0.0)
- npm

### Installation
```bash
npm install
```

### Development
To start the development server:
```bash
npm run dev
```
